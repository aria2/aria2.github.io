aria2 は 軽量で 複数サーバーから分割ダウンロードが可能なコマンドライン ダウンロード ユーティリティです。HTTP/HTTPS, FTP, SFTP, BitTorrent, Metalink に対応しています。 また、aria2 は JSON-RPC と XML-RPC インターフェースを搭載していますので、 aria2 をリモートコントロールすることができます。

ダウンロード

version 1.35.0 をダウンロード。 ソースコード, OS X バイナリ, Windows バイナリおよび Android バイナリがダウンロードできます。

1.19.0 以前のリリースはこちら.

フィーチャー

  • 分割ダウンロード. aria2 は複数のサーバー / プロトコルからファイルをダウンロードできます。 回線速度を有効に使った高速なダウンロードを実現します。

  • 軽量. aria2 はメモリや CPU 資源を大きく要求しません。ディスクキャッシュがオフの時、HTTP/FTP ダウンロードでのメモリ使用は通常 4MiB、BitTorrent ダウンロードで 9MiBとなります。また、CPU 使用率は 2.8MiB/sec のBitTorrentダウンロードで 6% 前後となります。

  • BitTorrent. 必要な機能がすべてそろった BitTorrent プロトコルが実装されています。例えば、DHT, PEX, Encryption, Magnet URI, Web-Seeding, ファイル選択ダウンロード, ローカルピア検出, UDP トラッカー などが実装されています。

  • Metalink サポート. aria2 は The Metalink Download Description Format (aka Metalink v4), Metalink version 3 および Metalink/HTTP. をサポートしています。Metalink は ファイル検証, HTTP/FTP/SFTP/BitTorrent からの同時ダウンロード, および 言語, 場所, OS などを指定する機能を提供します。

  • リモートコントロール. aria2 は RPC インターフェースに対応しており、aria2 プロセスをコントロールすることができます。JSON-RPC (over HTTP and WebSocket) と XML-RPC の2つのインターフェースに対応しています。

関連プロジェクト

  • apt-metalink: Debian/Ubuntu 向けに、より高速なパッケージのダウンロードを提供。
  • powerpill: 同時, 並列ダウンロードのための Pacman ラッパー。
  • python3-aria2jsonrpc: Aria2のJSON RPC インターフェースのラッパークラス。
  • aria2.js: aria2 RPCのためのJavaScript (ブラウザ及びNode.js) ライブラリとCLI。

UI フロントエンド

  • webui-aria2: aria2c 用のWebブラウザ インターフェース (2012 GSOC project)
  • uGet: Linux向けのダウンロード マネージャ
